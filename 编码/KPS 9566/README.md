# KPS 9566 系列
## 所收编码
### 编码形式
- KPS 9566
- EUC-KP

### 实际存在
- KPS 9566-2011

## 解说
KPS 9566-2011 转换表取自红星 3.0 的 /usr/lib/gconv/libKPS.so。该档案中 0x20520\~0x3011F 为 0x8100~0xFEFF 对应的 UTF-16 LE。

「柿」在官方映射中误映射为「杮」，原因不明。

## 字节结构
KPS 9566 如下：

|字节数|第一字节|第二字节|码位数|注释|
|-|-|-|-|-|
|单字节|0x00\~0x7F||128||
|双字节|0x81\~0xFE|0x41\~0x5A,0x61~0x7A,0x81\~0xFE|22428|实际使用 20594 个。|
